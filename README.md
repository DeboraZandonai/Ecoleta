**Note: This only needs to be done once for everyone.**

### Clone the repository to have a copy of the code on your machine

```bash
$ git clone https://github.com/DeboraZandonai/Ecoleta.git
```

### Navigate to the web project folder

```bash
$ cd Ecoleta/web
```

### Load dependencies

```bash
$ yarn
```

# Run the project

```bash
$ yarn start
```
